# Git
Comandos Básicos  

Clonar o repositório do github

git clone link

Listar o status do repositório Local

git status

Adicionar o arquivo

git add nome-arquivo

git add . adiciona todos

Realizar um commit 

git commit -m 'mensagem aqui'

Enviar arquivos ao Repositório 

git push 

Atualizar o repositório local com o github

git pull
